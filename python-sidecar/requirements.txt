# INSTALAÇÃO RECOMENDADA (Windows, PowerShell):
#   python -m venv venv
#   .\venv\Scripts\Activate.ps1
#   pip install --upgrade pip
#   pip install torch torchaudio torchvision --index-url https://download.pytorch.org/whl/cu126
#   pip install -r requirements.txt
#   pip install "audio-separator[gpu]>=0.44.0"   # [cpu] se nao tiver GPU NVIDIA
#
# NOTA: não fixar a versão do torch - deixe o pip resolver a versão mais
# recente compatível com o Python instalado e com o build CUDA escolhido
# (aqui usamos cu126, correspondente ao driver testado). Fixar versões
# antigas quebra em Python 3.12+, que não tem wheels para elas.
#
# Verifique antes: `nvidia-smi` deve mostrar um driver CUDA recente.
# Depois de instalar, confirme com:
#   python -c "import torch; print(torch.cuda.is_available())"
# Se retornar False, revise o driver CUDA antes de continuar (o pipeline
# foi pensado para rodar em GPU).

# --- Download / extração de áudio ---
yt-dlp>=2024.4.9

# --- Separação vocal ---
demucs>=4.0.1
# audio-separator: 2º passe de separação POR CIMA do stem de vocais do
# Demucs, pra isolar a voz PRINCIPAL de vozes de apoio/coro (o Demucs só
# separa vocal-do-instrumental, não lead-de-apoio). Validado com teste real
# ("Ama De Mi Sol", 13/07/2026): sem isso, coro sobreposto ao vocal
# principal faz o Whisper "escutar" palavras que não existem, o que
# envenena o alinhamento ao redor. Ver pipeline/separate.py:isolate_lead_vocal.
# Baixa modelos sob demanda (~900MB no 1º uso) - não-fatal se falhar (cai
# pro stem combinado do Demucs, ver main.py).
# ATENÇÃO: o pacote base NÃO instala o onnxruntime, que ele importa no topo
# do módulo - sem o extra [gpu] (ou [cpu]) o resgate falha silenciosamente.
# Extras não dão pra condicionar por máquina num requirements estático, então
# o setup-sidecar.ps1 instala o extra certo depois deste arquivo; em install
# manual, rode também: pip install "audio-separator[gpu]>=0.44.0" ([cpu] sem GPU).
audio-separator>=0.44.0
